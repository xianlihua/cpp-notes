## 算术运算符及计算过程的类型转换

**算数运算符的优先级**

  - \* / % 的优先级是相同的，且高于 + - 运算符
  - 两个运算数或数值表达式进行算术运算，其最终的数据类型取决于两个数的类型中谁的上限容量更大(个人理解)

**思考**

```c++
short s; // 16 bit
int i; // 32 bit
float f; // 32 bit
double d; // 64 bit (取决于编译器)
unsigned u; // 32 bit > int

// 下面表达式最终返回值的类型应该为 double
123 % s + (i + '@') + i * u – f / d
```
