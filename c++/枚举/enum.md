## 枚举的基本思想

枚举是利用已有知识进行答案猜测的一种问题求解的方法。

枚举的思想就是猜测。

从可能的集合中一一列举各元素，根据问题给定的检验条件判定哪些是成立的，能够使条件成立的就是问题的解。

### 枚举中的三个关键问题

- 给出解空间，建立简洁的数学模型
- 减少搜索的空间
- 采用合适的搜索顺序

### 实例：百钱买白鸡

> 鸡翁一值钱五, 鸡母一值钱三, 鸡雏三值钱一.
> 百钱买百鸡, 问鸡翁, 鸡母, 鸡雏各几何

```c++
#include <cstdio>
int main() {
    int x, y, z;
    for (x = 0; x < 20; x++) {
        for (y = 0; y < 33; y++) {
            z = 3 * (100 - 5 * x - 3 * y);
            if (z == 100 - x - y) {
                printf("%d %d %d\n", x, y, z);
            }
        }
    }
    return 0;
}
```
